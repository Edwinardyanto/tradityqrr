Pengaturan Pixel
Meta Pixel
Dengan mengisi Pixel ID dan Access Token Meta kamu, setiap pembelian yang terjadi lewat link referral kamu akan otomatis tercatat sebagai konversi di akun Meta Ads kamu.

Cara dapat Access Token:
1. Buka business.facebook.com/events_manager2
2. Pilih pixel kamu → tab Settings
3. Scroll ke Conversions API → Generate access token
Angka 15–16 digit dari Events Manager
Dari tab Settings pixel → Conversions API → Generate access token
Google Ads
Dengan mengisi Google Ads Conversion ID dan Label, setiap kali ada yang buka halaman checkout lewat link kamu, pixel Google Ads kamu akan aktif untuk remarketing dan audience building.

Cara dapat Conversion ID & Label:
1. Buka ads.google.com
2. Goals → Conversions → + New conversion action → Website
3. Pilih kategori Purchase → isi nama → Save
4. Pilih "Install the tag yourself"
5. Copy Conversion ID (format: AW-XXXXXXXXX) dan Conversion Label
Format: AW-XXXXXXXXX
String pendek yang didapat bersama Conversion ID
